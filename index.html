Fraud Detection Model

Tech Stack: Python, Jupyter Notebook, Pandas, Scikit-learn
Dataset: Kaggle: Fraud Detection Dataset by Kartik2112

This project explores how machine learning can be used to detect fraudulent credit card transactions based on structured behavioral features.

  • Cleaned and prepared a dataset of 100,000 anonymized transactions
  • Trained and compared Logistic Regression and Decision Tree classifiers
  • Evaluated performance using accuracy, recall, and F1-score
  • Visualized class imbalance and performance metrics with plots
  • Summarized analysis and findings in a visual PDF report
